GRadar Plugin

Das GRadar Plugin ist immer noch in Entwicklung, obwohl der Entwickler bereits vor längerer Zeit über 1.000,- Dollar an Spenden für den Kauf eines Radar-Gerätes eingesammelt hat. Es soll jedoch soweit laufen. Da ich selbst kein Radar-Gerät habe, kann ich das weder bestätigen noch selbst zur Entwicklung beitragen. Das Plugin startet jedoch auf Mac OS X, will aber die Aktivierung von OpenGL haben. Nähere Informationen dazu kann man unter dieser Datei finden: gradar.zip





Sollte es Anwender geben, die ein entsprechendes digitales Radargerät haben, bin ich gerne bereit, bei der Entwicklung mitzuhelfen. Das GRadar Plugin ist zwar nur für Garmin Geräte, jedoch gibt es vergleichbare Projekte für Radar-Antennen anderer Hersteller. Grundsätzlich ist wohl nur TCP/IP Router an Bord erforderlich, der das Ethernet-Kabel der Antenne aufnimmt und an den Computer weiterleiten kann. WireShark zur Beobachtung des Datenstroms ist eine wertvolle Hilfe bei der weiteren Entwicklung, da es von Hersteller zu Hersteller Unterschiede gibt.

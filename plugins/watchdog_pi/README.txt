Watchdog Plugin for OpenCPN
=======================================
Implement watchdog ability for opencpn

Compiling
=========
You have to be able to compile OpenCPN itself - Get the info at http://opencpn.org/ocpn/developers_dogual

* go to the plugin subdirectory in your OpenCPN source tree
* git clone git://github.com/seandepagnier/watchdog_pi.git

Build as normally:

* cd ..
* cd build
* cmake ..
* make
* sudo make install

* make package (to build deb, or installers for windows or mac)
